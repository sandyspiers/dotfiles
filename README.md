# Sandy's Dotfiles

> chaos, configured chaotically

## Install

```bash
./install all
```

## Gnome

```bash
./install gui
./install gnome
```

Use _Extension Manager_ to manage Gnome.
Start with 'Dash to Dock' and 'User Themes'.
Check [here](https://zebreus.github.io/all-gnome-backgrounds/) for wallpaper options.
